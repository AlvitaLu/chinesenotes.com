This inscription was written by Yan Zhenqing 颜真卿 in 779 in regular script 楷书. The inscription is in memory of Yan Zhenqing's forebears, in particular, his grandfather. Three of the four sides of the inscription are still remaining. It contains 39 lines with 38 characters on each line and 5 lines with 37 characters with a total of 1667 characters. It is 175×90×22 cm in size.

  • Page I Scan, SVG Trace： 名聞。七爲憲官，九爲省官，荐爲節度採
  • Page II Scan SVG Trace： 訪觀察使，魯郡公。允 臧，敦實有吏能，舉
  • Page III Scan, SVG Trace： 縣令，宰延昌。四爲御史，充太尉郭子儀
  • Page IV Scan, SVG Trace： 判官、江陵少尹、荆南行軍司馬。長卿、晉
  • Page V Scan, SVG Trace： 卿、邠、充國、質、多無祿早世。卿、倜、估
  • Page VI Scan, SVG Trace： 伋倫，並爲武官，玄孫紘，通義尉，沒于蠻
  • Page VII Scan, SVG Trace： 泉朙，孝義有吏道，父開土門佐其 謀彭
  • Page VIII Scan, SVG Trace：州司马。威明，邛州司马。季明、子干、沛、翊（州司馬。威明，邛州司馬。季明、子幹、沛、翊）
  • Page IX Scan, SVG Trace：颜，泉明男诞，及君外曾孙沈盈、卢逖，并（顏，泉明男誕，及君外曾孫沈盈、盧逖，並）
  • Page X Scan, SVG Trace：为逆贼所害，俱蒙赠五品京官。濬，好属（為逆賊所害，俱蒙贈五品京官。濬，好屬）
  • Page XI Scan, SVG Trace：文。翘、华、正，<岳页>并早夭。颖，好五言，挍书郎（文。翹、華、正，<嶽頁>並早夭。穎，好五言，挍書郎）
  • Page XII Scan, SVG Trace：颋仁孝方正，明经大理司直，充张万顷（頲仁孝方 正，明經大理司直，充張萬頃）
  • Page XIII Scan, SVG Trace：岭南营田判官。顗，凤翔参军。頍，通悟颇（嶺南營田判官。顗，鳳翔參軍。頍，通悟頗）
  • Page XIV Scan, SVG Trace： 善隷书，太子洗马、郑王府司马。并不幸（善隷書，太子洗馬、鄭王府司馬。並不幸）
  • Page XV Scan, SVG Trace： 短命。通明，好属文，项城尉。岁，温江丞。觌（短命。通明，好屬文，項城尉。歲，温江丞。覿）
  • Page XVI Scan, SVG Trace： 绵州参军。靓，盐亭尉。颢，仁和有政理，蓬（綿州參軍。靚，鹽亭尉。顥，仁和有政理，蓬）
