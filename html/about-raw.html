About the Project

The Fo Guang Shan Nan Tian Institute (NTI) Chinese Notes web site is a non-profit, open source project for use in learning Chinese, esepcially reading, and analyzing Chinese texts. Dislaimer: The Chinese Notes web site has no official affiliation with Nan Tian Institute and takes no responsibility for the accuracy of the content.

Check out the What's New page for updates.

Goals

The goals the project are

  1. Create an online dictionary that is easy to use for everybody interested in Chinese language, including casual readers, students, translators, and academics.
  2. Create electronic tools that are useful for lingustic analysis of literary Chinese texts, including identification of specialist historic terms.
  3. Share the source code and content with the general public. If you want to reuse source code or contribute source code or data then the GitHub project is built for you.

For more about the dictionary see Dictionary Design.
