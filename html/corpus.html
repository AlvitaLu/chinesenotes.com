The Chinese Notes software and dictionary is a platform that can be to create readers for multiple corpora. It is currently used at ntireader.org for a reader for the Taishō Shinshū Daizōkyō version of the Chinese Buddhist canon and at hbreader.org for a reader for Venerable Master Hsing Yun's collected writings. It is a simple system that any web developer can use to create a Chinese web reader from a set of plain text files. See the GitHub project for more details.
